# Game Site

Небольшой сайт с мини-играми, React/Vite-фронтендом и Express-бэкендом для таблиц рекордов.

## Игры

- `Clicker` - кликер с очками, монетами, автокликерами и достижениями.
- `Match3` - поле с заменой соседних элементов и сбором совпадений.
- `Longcat` - головоломка про заполнение поля длинным котом.
- `Match-Path` - уровни с соединением пар на сетке.
- `Number Merge` - сдвиги и объединение числовых плиток.
- `Snake` - классическая змейка со стартом, рестартом, счётом, уровнем и скоростью.
- `Keyboard Invaders` - тренажёр печати с волнами фраз, промахами, точностью и скоростью.

## Как открыть

На стартовом экране есть карточки выбора игры:

- `Clicker`
- `Match3`
- `Longcat`
- `Match-Path`
- `Number Merge`
- `Snake`
- `Keyboard Invaders`

## Управление

### Snake

- Стрелки или `WASD` - движение.
- `Space` - пауза.
- Стены смертельны: выход за поле завершает игру.

### Keyboard Invaders

- Печатайте первую живую букву слева направо.
- `P` - пауза.
- Пробел (`Space`) считается обычным символом фразы.
- Есть переключатель языка фраз: `EN` / `RU`.
